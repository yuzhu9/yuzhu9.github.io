发表于 2021-01-03 更新于 2021-01-04 分类于 笔记

CSS盒模型是什么

在CSS中，把所有HTML元素都看作盒子。HTML元素就是一个一个的盒子，而每一个盒子是由4个部分组成的，分别是：

  • content-box：显示内容

  • padding-box：内边距，包围在内容区域外部的空白区域

  • border-box：边框，包裹内容和内边距

  • margin-box：最外面的区域，是元素和其他元素之间的空白区域

盒模型分两种类型

content-box和border-box，也分别被称为标准盒模型和IE盒模型。一个盒子它是表现为content-box还是border-box，可以通过box-sizing属性来设置

content-box和border-box的区别是对元素设置width和height的表现不同：

  • content-box：width只代表content的width，而该元素的实际的宽度=width+左右padding+左右border；height的值只代表content的height，该元素在页面实际高度=height+上下padding+上下border
  • border-box：元素的实际宽度=width，width已经包括了content、左右padding和左右border；元素实际高度=height，height已经包括了content、上下padding和上下border

盒模型相关的属性

  • width：如上文所述，对于content-box，width是指content的宽度；对于border-box，width指content+左右padding+左右border的总和
  • height：对于content-box，height是指content的宽度；对于border-box，height指content+上下padding+上下border的总和
  • padding：有四个方向的值可以设定，padding-top/padding-right/padding-bottom/padding-left
  • border：有四个方向的值可以设定，border-top/border-right/border-bottom/border-left
  • margin：有四个方向的值可以设定，margin-top/margin-right/margin-bottom/margin-left，margin不会计入盒子实际大小，但它会影响盒子在页面所占的空间，影响的是盒子外面的空间。
