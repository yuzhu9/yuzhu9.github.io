2.浏览器的渲染过程

简单来说，有以下步骤

  1. 根据HTML构建HTML树（DOM）

  2. 根据CSS构建CSS树（CSSOM）

  3. 将两棵树合并成一棵渲染树（render tree）

  4. Layout布局（文档流、盒模型、计算大小和位置等）

  5. Paint绘制（把边框颜色、文字颜色、阴影等画出来）

transform的简单介绍

  • 四个常用功能：
    1. 位移translate
    2. 缩放scale
    3. 旋转rotate
    4. 倾斜skew
    5. 其他查看MDN

一般都需要配合transition过渡

inline元素不支持transform，需要先变成block

  1. translate常用写法(<length-percentage>表示一个值，该值可以是<length>或<percentage>)

    • translateX(<length-percentage>)
    • translateY(<length-percentage>)
    • translate(<length-percentage>,<length-percentage>)
    • translateZ(<length>)且父容器设置perspective
    • translate3d(x,y,z)

    Translate(-50%,-50%)可做绝对定位元素的居中

  2. scale常用写法

    • scaleX(<number>)
    • scaleY(<number>)
    • scale(<number>,<number>)

    较少使用，因为容易出现模糊

  3. rotate常用写法

    • rotate([<angel>|<zero>])
    • rotateZ([<angel>|<zero>])
    • rotateX([<angel>|<zero>])
    • rotateY([<angel>|<zero>])
    • rotate3d

    一般用于360度旋转制作loading

  4. skewMDN

  5. transform多重效果

    • transform: scale(1.2) translate(-100%,-100%)组合使用
    • transform: none；取消所有

transition 过渡

作用是补充中间帧

语法等查看transition过渡笔记

并不是所有属性都能过渡

  • display: none到block无法过渡

  • 一般改成visibility: hidden到visible

  • background颜色可以过渡

  • opacity可以过渡

    过渡必须要有起始，一般只有一次动画或者两次，如果除了起始状态，还有中间状态，有两种方法

  • 使用两次transform

    .a===transform===>.b

    .b===transform===>c

    用setTimeout或者监听transitionend事件

  • 使用animation

    声明关键帧

    添加动画

    如果要让动画停在最后一帧，在animation属性值里加forwards

    animation动画

    animation：时长|过渡方式|延迟|次数|方向|填充模式|是否暂停

  • 时长：1s或1000ms

  • 过渡方式：跟transition取值一样，如linear

  • 次数：3、2.4、infinite

  • 方向：reverse|alternate|alternate-reverse

  • 填充模式：none|forwards|backwards|both

  • 是否暂停：paused|running

  • 以上所有属性都有对应的单独属性

  • 更多内容查看animation-MDN
