JavaScript对象笔记

发表于 2021-01-07 更新于 2021-01-08 分类于 笔记

定义

JavaScript对象是JavaScript里面最重要的数据类型。

它是键值对{key:value}的集合，是无序的数据集合

属性名和属性值

  1. 每一个键名(key)又称为对象的属性名（property）

    • 对象的键名都是字符串（ES6引入Symbol值也可以作为键名）
    • 引号可以省略（比如标识名、数字），但就算省略了，键名也还是字符串
    • 如果键名不符合标识名的条件且不是数字，必须加引号，否则会报错

  2. 属性名（key）对应的值（value）称为对象的属性值

    • 属性值（value）可以是任何数据类型，可以是字符串、数字、也可以是对象
    • 如果一个属性的值为函数，通常把这个属性称为“方法”

原型

  • 每个对象都有原型
    • 原型里存着对象的共有属性
    • 比如obj={}obj的原型就是一个对象
    • obj.__proto__存着这个对象的地址
    • 这个对象里有toString / constructor / valueOf 等属性
  • 对象的原型也是对象
    • 对象的原型也有原型
    • obj={}的原型就是所有对象的原型
    • 这个原型包含所有对象的共有属性，是对象的根
    • 这个原型也有原型，是null
