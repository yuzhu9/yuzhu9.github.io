CSS（Cascading Style Sheets），层叠样式表

CSS历史

CSS是由哈肯·维姆·莱于1994年提出，目前使用最广泛的版本是CSS2.1，最新版本是CSS3，CSS3分模块进行升级

CSS层叠是指什么

层叠是指：

  • 样式层叠：可以多次同同一个选择器进行样式声明
  • 选择器层叠：可以用不同选择器对同一个元素进行样式声明
  • 文件层叠：可以用多个文件进行层叠
  • 这些特性使得CSS极度灵活

CSS语法

  • 语法一：样式语法

    • 所有符号都是英文符号
    • 区分大小写
    • 没有//注释
    • 任何地方写错了，都不会报错，浏览器会直接忽略从来产生bug

  • 语法二：at语法

    • @charset 必须放在第一行
    • 前两个at语法必须以分号;结尾
    • @media语法会单独教学
    • charset是字符集，但UTF-8是字符编码encoding

  • overflow溢出

    当内容大于容器

    • 内容的宽度或高度大于容易的，会溢出
    • 可用overflow来设置是否显示滚动条
    • auto是灵活设置
    • scroll是永远显示
    • hidden是直接隐藏溢出部分
    • visible是直接显示溢出部分
    • overflow可以分为overflow-x和overflow-y

CSS调试

  • 使用编辑器看颜色，如VSCode，不是很好用
  • 使用WebStorm看颜色
  • 使用开发者工具看警告
  • border调试法
    • 怀疑某个元素有问题，就给该元素加border
    • 如果border没出现，说明选择器或者语法错了
    • 如果border出现了，看看边界是否符合预期
    • bug解决了再把border删掉
  • 拼写错误是最常遇到的错误
