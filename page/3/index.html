CSS继承属性小结

发表于 2017-05-16 分类于 笔记

继承

继承是CSS的一个主要特征，它是依赖于祖先-后代的关系的。继承是一种机制，它允许样式不仅可以应用于某个特定的元素，还可以应用于它的后代。
元素的CSS属性分为继承性属性和非继承性属性。这决定了当你没有为元素的属性指定值时该如何计算值。

  • 当元素的继承属性没有指定值时，则取父元素的同属性的计算值。
  • 当元素的非继承属性没有指定值时，取属性的初始值。
  • inherit关键字用于显示地指定继承性，可用于继承性属性和非继承性属性。
