HTML常用标签复习笔记

发表于 2018-04-29 分类于 笔记

根元素

  • <html>根元素，其他所有元素必须是此元素的后代

元数据

  • <link>指定了外部资源与当前文档的关系。常用来链接样式表
  • <meta>表示那些不能由其它HTML元相关元素 (<base> <link> <script> <style> <title>) 之一表示的任何元数据信息
  • <style>包含了文档的样式化信息或者文档的一部分，该标签的样式信息通常是 CSS 的格式。
  • <title>定义文档的标题，显示在浏览器的标题栏或标签也上。它只可以包含文本。

内容分区

内容分区元素允许将文档内容从逻辑上进行组织划分。

  • <header>表示一组引导性的帮助，可能包含标题元素，也可以包含其他元素。如 logo等
  • <nav>导航栏
  • <h1>~<h6>标题标签
  • <section>表示文档中的一个区域，比如内容中的一个专题组
  • <footer>表示最近一个章节内容或者根节点元素的页脚。一个页脚通常包含该章节作者、版权数据或文档相关的链接等信息
  • <aside>表示一个和其余页面内容几乎无关的部分，可以被单独拆分出来而不会使整体受影响。通常表现为侧边栏或嵌入内容

文本内容

  • <blockquote>块级引用元素，代表其中的文字是引用内容。通常在渲染时，这部份内容会有一定的缩进。如果引文来源于网络，则可以将原内容的出处URL地址设置到 cite特性上
  • <main>呈现文档或应用的主体部分
  • <pre>表示预定义格式文本
  • <div>通用型的容器，可以用来对其他元素进行分组，语义上不代表任何特定类型的内容
  • <p>表示文本的一个段落
  • <hr>表示段落级元素之间的主题转换，表现为水平线。但目前被定义为语义上的，而不是表现层面上
  • <dl>表示一个包含术语定义以及描述的列表。通常用于展示词汇表或元数据（键-值对列表）
  • <dt>用于在定义列表中声明一个属于。该元素仅能作为<dl>的子元素。通常后面后跟着<dd>元素
  • <dd>用来指明一个描述列表元素中一个术语的描述。只能作为<dl>的子元素出现，并且必须跟着一个<dt>元素
  • <ol>有序列表，通常渲染为带有编号的列表
  • <ul>无序列表，表示无数值排序项的集合
  • <li>表示列表里的条目。它必须被包含在<ol>或者<ul>或者<menu>里

内联文本语义

  • <a>可以创建到其他网页、文件、同一页面内的位置、电子邮件地址或者任何其他URL的超链接
  • <b>浏览器中显示为粗体，但不起强调作用
  • <br>生成一个换行符号
  • <cite>表示一个作品的引用
  • <code>呈现一段计算机代码。默认情况下，它以浏览器的默认等宽字体显示
  • <em>标记需要用户着重阅读的内容，该元素是可以嵌套的，嵌套层次越深，包含的内容被认定为越需要着重阅读
  • <i>用于表现因某些原因需要区分普通文本的一系列文本，它的内容通常以斜体显示
  • <kbd>用于表示用户输入，它将产生一个行内元素，以浏览器的默认等宽字体显示
  • <q>用来引用短的文本
  • <small>该元素使文本的字体变小一号。在 HTML5 中，除了它的样式含义，这个元素被重新定义为表示边注释和附录细则，包括版权和法律文本
  • <span>通用行内容器，没有任何特殊语义
  • <strong>表示文本十分重要，一般用粗体显示
  • <sub>下标
  • <sup>上标
  • <time>用来表示24小时制时间或者公历日期，若表示日期则也可包含时间和时区
  • <u>使文本在其内容的基线下呈现下划线

图片和多媒体

  • <area>在图片上定义一个热点区域，可以关联一个超链接。该元素仅在‘<map>元素内部使用
  • <audio>表示音频内容，可以包含多个音频资源，这些音频资源可以使用src属性或者<source>元素来进行描述，浏览器将会选择最合适的一个来使用
  • <img>图像
  • <map>与<area>一起使用来定义一个图像映射（一个可点击的链接区域）
  • <track被当作媒体元素<audio> <video>的子元素来使用。它允许指定计时字幕（或者基于事件的数据），例如自动处理字幕
  • <video>用于嵌入视频内容

脚本

  • <canvas>可被用来通过脚本（通常是 JavaScript）绘制图形
  • <noscript>如果页面上的脚本类型不受支持或者在浏览器中关闭了脚本，使用该元素定义脚本未被执行时的替代内容
  • <script>用于嵌入或引入可执行脚本

编辑标识

  • <del>表示一些被从文档中删除的文字内容
  • <ins>表示已经被插入文档中的文本

表格内容

  • <caption>表示表格的标题，常常作为<table>元素的第一个子元素出现
  • <col>定义表格中的列，并用于定义所有公共单元格上的公共语义。它通常位于<colgroup>元素内
  • <colgroup>用来定义表中的一组列表
  • <table>表示表格数据
  • <tbody>表格的主体内容
  • <td>表格数据单元
  • <tfoot>定义表格中各列的总和或总结的行
  • <thead>定义表格的列头的行
  • <th>定义表格内的表头单元格
  • <tr>定义表格中的行
