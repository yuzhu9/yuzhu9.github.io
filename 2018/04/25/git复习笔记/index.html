git复习笔记

发表于 2018-04-25 分类于 笔记

1.什么是 Git

Git 是目前最好用的分布式版本控制系统，它的创作者是Linux之父林纳斯。详见Git 维基百科

2.什么是 GitHub

GitHub 是一个提供Git版本库托管的网站，是目前世界上最大的代码存放网站和开源社区。

3.最常见命令

git init 初始化一个空的 Git 版本库。

举栗子说明：例如我在桌面test目录下新建了一个叫gitlearning的目录，并跳转到这个目录，在此目录下输入git init命令就在gitlearning目录下创建了一个空的版本库。此时，用ls -a命令就可以查看出现了一个.git目录

git add将工作区的改动提交到暂存区。

例如我新建一个叫 index.html 的文件，使用git add index.html命令就可以将这个变动提交到暂存区。

git commit将暂存区的内容提交到 git 仓库

例如将之前添加到暂存区的index.html提交到仓库。在上一步git add index.html之后git commit就可以把暂存区的内容提交到仓库。
