CSS常见布局学习笔记

发表于 2018-06-15 分类于 笔记

1.左右布局

总得来说，可以使用的方法有浮动、定位和 flex 布局

  • 左右宽度都固定

  • 左侧宽度固定，右侧自适应

  • 右侧宽度固定，左侧自适应

2.左中右布局

  • 宽度均固定

  • 左右固定，中间自适应

3.水平居中

  • Inline-block 元素或文本对其父元素使用text-align: center属性即可
  • block 元素
    • 1.使用：margin-left: auto;margin-right: auto;
    • 2.对父元素用 flex 布局：display: flex;justify-content: center
    • 3.使用定位方法：父元素position: relative; 子元素positio: absolute;left: 50%;transform: translateX(-50%);

4.垂直居中

  • 单行文本：对其父元素设置一样的padding-top和padding-bottom值即可，也可以将父元素的高度和行高设置为相同
  • 单行 inline-block 元素：对该元素设置vertical-align: top，对其父元素设置相同的padding-top和padding-bottom值
  • block 元素
    • 1.使用定位方法：父元素position: relative; 子元素positio: absolute;top: 50%;transform: translateY(-50%);
    • 2.对父元素用 flex 布局：display: flex;flex-direction: column;justify-content: center

其他小技巧

  • 1.能不写定宽度高度尽量不写固定宽高，通过内容来撑开
  • 2.尽量使用max-width属性来替代width属性
  • 3.为元素添加border属性是极为有效的找出问题的方法
