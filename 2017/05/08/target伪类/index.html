:target伪类

发表于 2017-05-08 分类于 笔记

相关概念

带有片段标识符的URI链接到文档中某一个元素 ，被称为是目标元素。:target 选择器可用于选取当前活动的目标元素。

简单示例

以下代码示例，当鼠标点击a标签，p标签的文字就会变成红色。因为id为demo的p元素是目标元素。当点击a标签时，目标元素p就是当前活动的目标元素，target伪类就会匹配到这个元素，设置的样式就会生效。

一些黑科技

  • 隐藏或显示元素

  • 侧边栏抽屉式导航

  • 弹出式模态框

  • 改变全局样式
