初识Flex布局

发表于 2017-05-06 分类于 笔记

Flex布局基本概念

1.Flex布局（flex layout）：弹性布局，是为了呈现复杂的应用与页面而设计出来的新的布局模式
布局比较适合Web应用程序的一些小组件和小规模的布局，而Grid布局更适合用于一些大规模的布局

2.一个设有display:flex或display:inline-flex的元素是一个Flex容器（flex container）。Flex容器的子元素被称为Flex项目（flex item）。设为Flex布局以后，子元素的float、clear和vertical-align属性将失效。下图解释了Flex布局的主要思想

容器默认存在两根轴：上图是水平的主轴(main axis)和垂直的侧轴(cross axis)。主轴的开始位置叫做main start，结束位置叫做main end；侧轴的开始位置叫做cross start，结束位置叫做cross end

  • 主轴（main axis）
    flex item沿着主轴(main axis)排列。main axis主要用来配置Flex项目，它不一定是水平，主要取决于flex-direction属性

  • 主轴起点(main start)、主轴终点(main end)
    flex item从容器的main start边开始，往main end边结束

  • 主轴长度(main size)
    flex item在主轴方向的宽度或高度就是main size，flex item的main size是width属性或height属性，由哪个对着主轴方向决定

  • 侧轴(cross axis)
    与main axis垂直的轴称作cross axis，是侧轴方向的延伸

  • 侧轴起点(cross start)、侧轴终点(cross end)
    伸缩行的配置从容器的cross start边开始，往cross end边结束。

  • 侧轴长度(cross size)
    flex item在侧轴方向的width属性或height属性就是flex item的cross size，由哪一个对着侧轴方向决定

  • Flex容器（flex container）

1.display: flex这个属性让元素产生块级flex container
2.display: inline-flex这个属性让元素产生行内级flex container

flex container的属性

  • flex-direction
    该属性决定主轴的方向（即flex item的排列方向），有4个值

  • flex-wrap
    该属性定义如何换行，有3个值

  • flex-flow
    该属性是flex direction和flex-wrap的简写形式，默认值是row nowrap

  • justify-content
    该属性定义flex item在主轴上的对齐方式，有5个值

  • align-items
    该属性定义flex item在侧轴上的对齐方式，有5个值

  • align-content
    该属性定义多根轴线的对齐方式。如果只有一根轴线，该属性不起作用，有6个值

    flex item的属性

  • flex-grow
    该属性定义flex item的放大比例（如果项目存在剩余空间），默认值为0（即不放大）。如果每一个flex item都设置了flex-grow属性的值，那么flex item将按照自身flex-grow的值/所有flex item的flex-grow的值加起来的总和的比例占据剩余空间

  • order
    该属性定义flex item的排列顺序。数值越小，排列越靠前，默认为0

  • flex-shrink
    该属性定义了flex item的缩小比例。默认为1，即如果空间不足，该flex item将缩小。如果所有flex tiem的flex-shrink属性都为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。

  • flex-basis
    该属性定义了在分配多与空间之前，flex item所占据的main size。浏览器根据这个属性，计算主轴是否有多余空间。默认值是auto，即flex item的本来大小

  • flex
    该属性是flex-grow，flex-shrink和flex-basis的简写，默认值为0 1 auto，后面两个值可选。该属性有两个快捷值auto（即1 1 auto）和none（即0 0 auto）

  • align-self
    该属性是设置单个flex item的对齐方式，可覆盖align-items属性，有6个值
