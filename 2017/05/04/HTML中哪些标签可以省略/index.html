HTML中哪些标签可以省略

发表于 2017-05-04 分类于 笔记

1.html标签

  • 可以省略起始标签<html>：html标签的内容不以注释作为开始
  • 可以省略结束标签</html>：结束标签后面没有紧跟注释

2.head标签

  • 可以省略起始标签<head>：1.head标签内容为空; 2.以一个标签作为开始
  • 可以省略结束标签</head>：head结束标签后面不是紧跟空白字符或者注释

3.body标签

  • 可以省略起始标签<body>：1.body标签内容为空； 2.不是以空白字符、注释、script元素、 style元素、meta元素、link元素、template元素为开头
  • 可以省略结束标签</body>：body结束标签后面的内容不是注释、空白字符、

4.tbody标签

  • 可以省略起始标签<tbody>：1.tbody标签内容不为空； 2.它是以tr标签开头，同时在它前面的不能是省略了结束标签的tbody、thead、tfoot
  • 可以省略结束标签</tbody>：1.tbody结束标签后面紧跟着tbody或tfoot标签； 2.当前tbody结束标签是父元素的最后一个子元素

5.li标签

  • 可以省略结束标签</li>：1.li结束标签后面的元素是li; 2.当前li结束标签是父元素的最后一个子元素

6.dt标签

  • 可以省略结束标签<dt>：dt结束标签后面紧跟着dt或dd标签

7.dd标签

  • 可以省略结束标签</dd>：1.dd结束标签后面紧跟着dd或dt标签； 2.当前dd结束标签是父元素的最后一个子元素

8.p标签

  • 可以省略结束标签</p>：1.p结束标签后面紧跟着address、article、aside、blockquote、details、div、dl、fieldset、figcaption、figure、footer、form、h1、h2、h3、h4、h5、h6、header、hr、main、menu、nav、ol、p、pre、seciton、table、ul元素； 2.当前p结束标签是父元素的最后一个子元素，并且父元素不是a、audio、del、ins、map、nosscript、video元素

9.rt标签

  • 可以省略结束标签</rt>：1.rt结束标签后面紧跟着rb、rt、rtc或rp标签； 2.当前rt结束标签是父元素的最后一个子元素

10.rtc标签

  • 可以省略结束标签</rtc>：1.rtc结束标签后面紧跟着rb或rtc标签； 2.当前rtc结束标签是父元素的最后一个子元素

11.rp标签

  • 可以省略结束标签</rp>：1.rp结束标签后面紧跟着rb、rt、rtc或rp标签； 2.当前rp结束标签是父元素的最后一个子元素

12.colgroup标签

  • 可以省略结束标签</colgroup>：当前colgroup结束标签后面不是紧跟着空白字符或注释

13.thead标签

  • 可以省略结束标签</thead>：thead结束标签后面紧跟着tbody或tfoot标签

14.tfoot标签

  • 可以省略结束标签</tfoot>：当前tfoot结束标签后面紧跟着tbody标签； 2.当前tfoot结束标签是父元素的最后一个子元素

15.tr标签

  • 可以省略结束标签</tr>：1.当前tr结束标签后面紧跟着tr标签； 2.当前tr结束标签是父元素的最后一个子元素

16.td标签

  • 可以省略结束标签</td>：1.当前td结束标签后面紧跟着td或th标签； 2.当前td结束标签是父元素的最后一个子元素

17.th标签

  • 可以省略结束标签</th>：1.当前th结束标签后面紧跟着td或th标签； 2.当前th结束标签是父元素的最后一个子元素

18.optgroup标签

  • 可以省略结束标签</optgroup>：1.当前optgroup结束标签后面紧跟着optgroup标签； 2.当前optgroup结束标签是父元素的最后一个子元素

19.option标签

  • 可以省略结束标签</option>：1.1.当前option结束标签后面紧跟着option或optgroup标签； 2.当前option结束标签是父元素的最后一个子元素

注：“空白字符”是指：空格(U+0020)、制表符(U+0009)、换行(U+000A)、回车(U+000D)、换页(U+000C)。
