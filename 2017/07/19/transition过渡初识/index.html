transition过渡初识

发表于 2017-07-19 分类于 笔记

CSS 3 过渡功能：CSS 3 的 transition 属性允许 CSS 的属性值在一定的时间区间内平滑地过渡。这种效果可以在鼠标单击、获得焦点、被点击或对元素任何改变中触发，并平滑地以动画效果改变 CSS 的属性值。

CSS 创建简单过渡的步骤

  1. 在默认样式中声明元素的初始样式状态。
  2. 声明过渡元素最终状态样式，比如悬浮状态。
  3. 在默认样式中通过添加过渡函数，添加一些不同的样式。

CSS 3 过渡子属性

在 transition 属性中包括四个子属性：

  • transition-property：指定应用过渡动画的 CSS 属性名称
  • transition-duration：指定过渡所需时间
  • transition-delay：指定过渡延迟时间
  • transition-timing-function：指定过渡函数

transition-property

指定应用过渡动画的 CSS 属性名称，语法 ：

none：表示没有过渡动画

all：默认值。表示所有可被动画的属性都表现出过渡动画

single-transition-property>：表示指定的样式，例如 width 、height 、background 等，可以同时指定不同的属性，属性之间用 ,隔开

transition-duration

指定过渡所需时间，语法 ：

time：表示时间的数值，单位为 s（秒）或 ms（毫秒），不能为负值；默认值为0，表示不会呈现过渡动画，属性瞬间完成转变。可以设置多个值，每个值之间用,分隔（设置多个过渡属性时才需要设置多个时间）

transition-delay

指定过渡延迟时间，语法 ：

time：表示动画过渡效果将在何时开始的数值，单位为 s（秒）或 ms（毫秒），可以是正整数、负整数和0，非零的时候必须设置单位。可以设置多个值，每个值之间用,分隔（设置多个过渡属性时才需要设置多个延迟时间）

  • 正整数：元素的过渡动作不会立即触发，当过了设定的时间值之后才触发
  • 负整数：元素的过渡动作会从该时间点开始显示，之间的动作被截断
  • 0 ：元素的过渡动作会立即触发，没有任何延迟。默认情况下，transition-delay 取值为0。

transition-timing-function

指定过渡函数，语法：

single-transition-timing-function：指单个过渡函数，过渡函数主要有以下这些：

  • ease：默认值，元素样式从初始状态过渡到终止状态时速度由快到慢，逐渐变慢
  • linear：元素样式从初始状态过渡到终止状态速度是匀速
  • ease-in：元素样式从初始状态到终止状态，速度越来越快，呈一种加速状态，这种效果称为渐显效果
  • ease-out：元素样式从初始状态到终止状态，速度越来越慢，呈一种减速状态，这种效果称为渐隐效果
  • ease-in-out：元素样式从初始状态到终止状态，先加速再减速。这种效果称为渐显渐隐效果
