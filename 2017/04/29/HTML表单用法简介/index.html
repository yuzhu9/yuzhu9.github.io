HTML表单用法简介

HTML表单用于收集用户输入，form元素定义HTML表单,form标签的action属性定义在提交表单时执行的动作，method属性规定在提交表单时所用的 HTTP 方法（get或post）。
<form action="#" mothod="get/post">

form主要包含以下表单元素：

(1)input元素，元素根据不同的type属性，可以变化为多种形态：

  • text单行文本输入框，例：
    用户名：<input type="text" name="username">

  • password密码输入框，字段中的字符会被做掩码处理（显示为星号或实心圆），例：
    密码：<input type="password" name="pwd">

  • radio单选按钮,同一组需要设置相同的name值，单选按钮允许用户在同一组选项中选择其中之一，通过添加checked属性来定义预定义选项，例：
    性别：<input type="radio" name="gender" value="male">男 <input type="radio" name="gender" value="female">女

  • checkbox多选按钮，通过添加checked属性来定义预定义选项，例：

  • button普通按钮，例：
    <input type="button" value="按钮">

  • submit提交表单数据的按钮，例：
    <input type="submit" value="提交">

  • reset重置按钮，点击后清空用户填写的未提交数据，例：
    <input type="reset" value="重置">

(3)textarea元素定义多行输入字段（文本域），例：

<textarea name="message">留言备注</textarea>

(4)button元素定义可点击的按钮，例：

<button>点击</button>
