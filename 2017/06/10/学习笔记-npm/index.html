学习笔记-npm

发表于 2017-06-10 分类于 笔记

什么是npm

npm全称Node Package Manager（Node包管理器），Node.js内置了npm。
详细内容

配置npm

  • npm config set loglevel http让你知道 npm 发的每一个请求
  • npm config set progress false关闭那个进度条
  • npm config set registry https://registry.npm.taobao.org/设置地址为淘宝镜像地址，以加快安装速度
    （想要恢复成原样，运行npm config delete registry即可）

发布代码

发布代码注意如果地址为淘宝镜像地址，运行npm config delete registry修改为原地址再发布

  • 1.转到在要发布的代码文件夹
  • 2.运行npm init初始化，得到package.json文件
  • 3.打开package.json文件，完善对应信息
  • 4.npm adduser填写npm用户名密码
  • 5.npm publish

安装包

下载安装可运行npm config set registry https://registry.npm.taobao.org/设置为淘宝镜像

  • 1.转到存放的文件夹
  • 2.npm install packageName``packageName即要下载的包名字
  • 3.安装前，npm install会检查node_modules是否有指定的包（如果有就不用重新安装，即使远程仓库已经有了一个新版本）
    如果要强制重新安装，可运行npm install <packageName> --force或npm install <packageName> --f

更新已安装的包

npm update <packageName>它会先到远程仓库查询最新版本，然后查询本地版本。如果本地版本不存在，或者远程版本较新，就会安装。
