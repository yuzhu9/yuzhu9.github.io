2.Array.prototype.every()

every()方法测试数组的所有元素是否都通过了指定函数的测试

every方法为数组中的每个元素执行一次 callback 函数，直到它找到一个使 callback 返回”假值”（表示可转换为布尔值 false 的值）的元素。
callback被调用时传入三个参数：元素的值，元素的索引，被遍历的数组。
every不会改变原数组。

数组numbers的第一个元素是1，它不是2的倍数，因此isEven函数返回false,然后every执行结束。

3.Array.prototype.some()

some()方法测试数组中的某些元素是否通过由提供的函数实现的测试。

some为数组中的每一个元素执行一次callback函数，直到找到一个使得callback返回一个“真值”（即可转换为布尔值true的值）。如果找到了这样一个值，some将会立即返回true。否则,some返回false。callback只会在那些”有值“的索引上被调用，不会在那些被删除或从来未被赋值的索引上调用。
callback被调用时传入三个参数：元素的值，元素的索引，被遍历的数组。
some 被调用时不会改变数组。

数组numbers中的第一个偶数是2（第二个元素），isEven函数返回true

4.Array.prototype.forEach()

forEach() 方法对数组的每个元素执行一次提供的函数。

forEach方法按升序为数组中含有效值的每一项执行一次callback 函数
callback 函数会被依次传入三个参数：元素的值，元素的索引，被遍历的数组。
没有办法中止或者跳出 forEach 循环，除了抛出一个异常。
forEach()为每个数组元素执行callback函数.它总是返回 undefined值，并且不可链式调用。

forEach方法为numbers数组的每一个元素调用了isEach函数，打印了所有的元素，并且它总是返回undefined值

5.Array.prototype.map()

map() 方法创建一个新数组，其结果是该数组中的每个元素都调用一个提供的函数后返回的结果。

map方法会给原数组中的每个元素都按顺序调用一次callback函数。callback每次执行后的返回值（包括undefined）组合起来形成一个新数组。
map方法不修改调用它的原数组本身

map方法创建了一个新数组，它的值是执行isEven的运行结果。

7.Array.prototype.reduce()

reduce()方法对累加器和数组中的每个元素 (从左到右)应用一个函数，将其减少为单个值。

reduce方法返回值是函数累计处理的结果
callback函数包含四个参数：
1.初始值accumulator（上一次调用回调返回的值，或者是提供的初始值（initialValue））
2.当前元素值currentValue
3.当前索引，如果提供了initialValue，从0开始，否则从1开始
4.调用reduce的数组

回调函数第一次执行时，accumulator和currentValue的取值有两种情况：

  • 1.调用reduce时提供initialValue：accumulator取值为initialValue，currentValue取数组中的第一个值
  • 2.没有提供initialValue：accumulator取数组中的第一个值，currentValue取数组中的第二个值

上面栗子中，reduce方法给了初始值0，因此第一次执行回调函数时，传入的参数就分别是初始值0和当前元素值0（数组中的第一个值0），返回值为这两个数的求和结果0；第二次执行时，传入的参数就是上一次的返回值0和当前元素值1（数组中的第二个值1），返回值是1；第三次执行时，传入的参数就是上次的返回值1和当前元素值2（数组中的第三个值2），返回值是3；第四次执行时，传入的参数是上次的返回是3和当前元素值3（数组中的第四个值3），返回值是6。

8.Array.prototype.join()

join()方法将数组（或一个类数组对象）的所有元素连接到一个字符串中。

所有的数组元素被转换成字符串，再用一个分隔符将这些字符串连接起来。如果元素是undefined 或者null， 则会转化成空字符串
join() 方法，不会改变数组
如果join方法不写参数，默认为逗号 “,”分隔
如果参数是空字符串(“”)，则所有元素之间都没有任何字符

9.Array.prototype.push()

push()方法将一个或多个元素添加到数组的末尾，并返回数组的新长度。

当调用该方法时，新的 length 属性值将被返回。

10.Array.prototype.unshift()

unshift()方法将一个或多个元素添加到数组的开头，并返回新数组的长度。

unshift 方法会在调用它的类数组（array-like）对象的开始位置插入给定的参数。

11.Array.prototype.pop()

pop()方法从数组中删除最后一个元素，并返回该元素的值。此方法更改数组的长度。

从数组中删除的元素; 如果数组为空返回undefined。

12.Array.prototype.shift()

shift()方法从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度。

从数组中删除的元素;如果数组为空，返回undefined。
