SS

科学上网，SS网站网站经常更换，购买服务后网站更换地址会把新地址用邮件到你注册的邮箱中，软件和设置方法一般都可以在SS网站上找到

  • （通用）安装软件，在SS网站下载软件，设置好代理，开启全局模式即可（缺点是访问国内网站速度变慢）

  • （通用）安装软件之后配合Chrome插件（既可以科学上网，又不影响访问国内网站的速度），步骤如下：

    1. 打开SS代理，开启全局模式

    2. 安装插件Proxy SwitchyOmega

    3. 下载配置文件OmegaOptions.bak，SS网站可下载

    4. 在Chrome插件栏右键选择选项，导入/导出，从备份文件恢复

    5. 插件设置为自动切换模式

    6. 插件设置好了自动切换之后，SS全局代理就可以禁止，SS代理修改为直连模式

  • 命令行科学上网

    • （Windows）Windows使用命令行科学上网的步骤

      1. 下载Proxifier
      2. 配置Proxifier，可以Gloogle搜索教程

    • （Mac）Mac使用命令行科学上网的步骤

      1. 安装homebrew，步骤见Homebrew

      2. 安装proxychains-ng

      3. 配置proxychains-ng

        1. 下载配置文件

        2. 添加bash alias

        3. source一下

        4. 输入pc git clone xxx 或者 pc brew install xxx –verbose，这个命令行就是翻墙的

  3. 安装插件Proxy SwitchyOmega，安装方法见SS

Cmder

（Windows）Windows系统好用的命令行工具Cmder，内置Git Bash，也支持Cmder和PowerShell

  1. 在Cmder官网下载，下载之后右键点击，勾选解除锁定，之后点击确定，再解压缩

  2. 在D/E/F任一盘新建目录，例如在D盘新建名为Software的目录，将解压之后的Cmder目录剪切过去

  3. 双击Cmder.exe打开它，接着进行配置，在General中设置Interface language为中文，勾选从屏幕顶部Quake向下滑动，在大小&位置中设置窗口大小为正常，宽度高度都设置为90%，保存设置

  4. 步骤3保存设置之后，关闭Cmder，重新打开，发现Interface language设置中文没有生效，点击General中右上角的Export导出配置文件，找出配置文件ConEmu.xml，用编辑器打开配置文件，搜索language，修改data的值为zh保存退出，设置中文界面成功，退出Cmder。

  5. 重新打开Cmder，设置字体为Fira Code

  6. 设置shell为{bash::bash}

  7. 其他设置根据需求设置，例如一键呼出、分屏、换肤，以及对应的快捷键配置等可搜索教程。

yarn

（通用）进入yarn官网根据提示选择相应的系统（Windows/Mac）进行安装，可以替代npm，相对更稳定，速度比npm快一点，bug少一点，安装好了之后命令行输入yarn --version可以看到版本号，输入yarn global add yrm安装源管理器yrm，yrm ls yrm use taobao使用淘宝源

VSCode

（通用）进入VSCode官网下载安装，常用配置如下：

  1. 汉化：打开VSCode点击扩展图标（即侧边最后一个图标），搜索chinese安装，关闭重开即生效
  2. 自动保存：文件-首选项-设置，搜索auto save，选择第三个选项onFocusChange
  3. 自动格式化：同上，搜索format on save 勾选即可（可能会格式化别人的代码）
  4. 把面板移到右边：查看-外观-向右移动侧边栏
  5. 隐藏菜单栏：查看-外观-显示菜单栏去掉勾选（可以按Alt键来显示菜单栏）
  6. 设置Fira Code字体：文件-首选项-设置，搜索字体，修改Font Family，修改为Fira Code,Consolas,monospace，还可以启用或禁用字体连字
  7. Code Spell Checker插件：防止代码里面单词拼错，安装方法是点击扩展，搜索code spell checker，安装之后拼写有问题的会有提示，可以移动鼠标到错误的单词上，右键快速修复就可以修复拼写错误问题，如果有些提示不是拼写错误，同样可以把鼠标移动到单词上，右键选择把单词加入用户词典或当前词典就不会提示（推荐加入当前词典）
  8. Git Easy插件：方便操作git，同上，搜索git easy安装
  9. Prettier插件：更好地格式化工具，方法是在扩展中搜索prettier安装

VSCode功能：

  • 开启终端：菜单栏点击终端，新建终端，VSCode界面就会显示控制台，可以选择新建、分屏、终止等功能

  • 搜索文件夹或文件：

    • 搜索文件夹：在侧边栏选择搜索（图标是放大镜的)输入要查找的单词，就会在搜索框下方显示出对应的文件，还可以进行替换操作。

    • 搜索当前文件：Ctrl + F输入要搜索的单词即可搜索，Ctrl + H替换，点击全部替换图标可以全部替换，如果只需要在特定选中的内容中替换，就先选中需要查找替换的内容区域，点击在选中的内容中查找图标，再点击全部替换

  • 设置语法：在界面右下角点击选择文件格式

  • Emmet快捷操作：例如多行操作，选中多行内容，用div标签包裹，先选中需要包裹的多行内容，按Ctrl + Shift + P输入emmet wrap，在显示列表中选中输入包裹个别行，再输入div*

  • Git操作

  • 调试JavaScript/TypeScript

  • 快捷键：

    • 快读查找Ctrl + P
    • 快速输入命令Ctrl + Shift + P
    • 多位置输入Alt + 单击
